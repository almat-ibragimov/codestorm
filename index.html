CodeStorm - это школа программирования,
направленная на развитие детей. Курс
фронтенд-разработки научит ваших детей
создавать собственные веб-сайты с помощью
языков HTML, CSS и JavaScript. Наша цель - это
создание нового поколения специалистов и
профессионалов в сфере IT.

Записаться

Как проходит обучение:

HTML

Первым делом ваши дети будут
учить язык гипертекстовой
разметки - HTML. С помощью
HTML создается вся логическая
структура сайта, и без него не
может существовать ни один
сайт! Наши ученики изучат все
необходимые теги и смогут также
писать логическую структуру
сайта.

CSS

После изучения HTML наши
ученики приступят к изучанию
каскадных таблиц стилей, то есть
CSS. Именно CSS отвечает за
внешний вид любого сайта. Ваши
дети изучат все необходимые
стили, научатся адаптировать
сайты под различные устройства
и смогут улучшать внешний вид
собственных веб-страниц.

JavaScript

Далее наши ученики приступят к
изучению языка
программирования JavaScript.
Если HTML и CSS создают макет
сайта, то JavaScript по сути
оживляет сайт. Наши ученики
научатся создавать
интерактивные веб-страницы и
смогут свободно писать код на
JavaScript!

Доступная цена курса

Оплата производится ежемесячно и в месяц данный курс будет стоить всего лишь 10.000 тенге!
Длительность курса составит примерно 6 месяцев. Наша цель - это предоставление знаний по
максимально доступной цене! Мы хотим, чтобы как можно больше людей начали понимать
программирование!

Удобный график уроков

Уроки проходят в субботу с 16:00 до 17:00, с 17:30 до 18:30 и с 19:00 до 20:00 по времени Алматы. Мы
планируем собрать 3 группы по 10 человек и для каждой группы уроки будут проходить 1 раз в
неделю. Подобный график уроков и вправду будет удобен для большинства подростков!

Приятный формат обучения

Google Meet будет основной платформой по проведению уроков, а задания с урока и домашние задания
будут отправляться в группу в WhatsApp или в Telegram. Ученики в случае чего смогут помогать друг-
другу, что в свою очередь будет способствовать развитию командной работы и дружеских отношений
между учениками!

Самое главное практика

Для каждого урока мы подготовили презентации с подробным объяснением тем и различными
заданиями, также в конце каждого урока будут даваться домашние задания. Подобный формат
преподавания полностью погружает ребёнка в тему урока! В оссобенности полное погружение будет
в конце каждого модуля, ведь именно тогда наши ученики будут разрабатывать собственные
проекты!

Индивидуальное обучение

Если вы хотите, чтобы ваш ребёнок обучался индивидуально, то в таком случае цена курса будет
составлять уже 15.000 тенге в месяц! Однако при таком формате обучения ребёнок будет лучше
понимать каждую новую тему. Уроки с индивидуальным обучением проходят в воскресенье с 10:00
до 21:00. Вы можете выбрать любое время в данном промежутке!
